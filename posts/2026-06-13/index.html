Arch Linux AUR Malware Attack#

Over 400 community-maintained packages in the Arch Linux User Repository (AUR) were compromised with malware. While official Arch Linux packages remain secure, the AUR’s trust model is now under scrutiny as maintainers work to ban affected accounts and purge malicious content.

  • https://www.phoronix.com/news/Arch-Linux-AUR-400-Compromised

Ladybird Browser Ends Public PRs#

The Ladybird browser project has ceased accepting public pull requests, citing the ease with which AI tools can now produce convincing but malicious or low-quality code. To ensure security and architectural integrity as they approach an alpha release, only project maintainers are now permitted to introduce code changes.

  • https://ladybird.org/posts/changing-how-we-develop-ladybird/

Linux 7.2 Hardware Support Expansion#

The upcoming Linux 7.2 kernel will bring significant hardware enablement, including the ability to boot on Apple M3 Macs and performance boosts via Cache Aware Scheduling. It also advances open-source AMD graphics drivers with initial HDMI 2.1 support and prepares for next-gen Intel and NVIDIA hardware.

  • https://www.phoronix.com/news/Linux-7.2-Features-Expected

Ubuntu 26.10 to Adopt Dbus-Broker#

Ubuntu 26.10 will switch its default D-Bus implementation to Dbus-Broker, a move that improves system performance and reliability. The transition was previously delayed by dependencies in GNOME’s GDM and Ubuntu’s AppArmor/Snap integration.

NTFS Linux Driver Symlink Improvements#

The new Linux NTFS kernel driver is being updated to support native Windows symbolic links, allowing for more transparent file system interaction between Linux and Windows native symlinks, moving beyond simple .lnk shortcuts.
