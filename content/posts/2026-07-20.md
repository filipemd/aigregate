## Linux GPU Network Offloading

Network packet processing is moving from the CPU to the GPU to bypass CPU bottlenecks. New Linux kernel patches enable AMD GPUs to handle high-speed packet processing and IPsec crypto directly in-kernel, significantly increasing throughput by leveraging the GPU's parallel architecture.

- [https://www.phoronix.com/news/KNOD-Network-Offload-AMD-GPUs](https://www.phoronix.com/news/KNOD-Network-Offload-AMD-GPUs)

## Qualcomm Crypto Driver Disabled

A dedicated hardware accelerator for encryption is being removed from default Linux builds because it is actually slower than the CPU it was meant to accelerate. The driver has a history of bugs and security race conditions, rendering it 'harmful' to system stability.

- [https://www.phoronix.com/news/Linux-QCE-Disabling](https://www.phoronix.com/news/Linux-QCE-Disabling)

## GNOME Security Policy Shift

GNOME is drastically shortening its vulnerability disclosure timeline to 30 days because AI-generated reports have become so common that non-AI reports are now 'moderately unusual.' The project is adapting its security workflow to handle the flood of automated vulnerability discoveries.

- [https://blogs.gnome.org/mcatanzaro/2026/07/20/some-changes-to-gnome-security-tracking/](https://blogs.gnome.org/mcatanzaro/2026/07/20/some-changes-to-gnome-security-tracking/)
