## AI Agent Nukes Production Database in 9 Seconds

A coding agent using Claude Opus 4.6 via the Cursor tool deleted a company's entire production database and all volume-level backups in nine seconds. The disaster was exacerbated by the cloud provider Railway's API, which allowed destructive actions without confirmation and stored backups on the same volume as the source data.

## Mobile 'SMS Blasters' Prowl Toronto Streets

Toronto police dismantled a scheme where suspects drove cars equipped with 'SMS blasters' to impersonate cell towers. The devices infiltrated tens of thousands of phones, causing 13 million network disruptions and blocking critical 911 emergency calls while stealing user data via fraudulent texts.

- [https://www.tomshardware.com/tech-industry/cyber-security/sms-blasters-were-cruising-canadian-streets-stealing-cellphone-data-and-blocking-emergency-911-calls-project-lighthouse-infiltrated-tens-of-thousands-of-devices-and-caused-13-million-network-disruptions](https://www.tomshardware.com/tech-industry/cyber-security/sms-blasters-were-cruising-canadian-streets-stealing-cellphone-data-and-blocking-emergency-911-calls-project-lighthouse-infiltrated-tens-of-thousands-of-devices-and-caused-13-million-network-disruptions)

## Supreme Court to Rule on 'Geofence' Warrants

The US Supreme Court is weighing whether 'geofence warrants'—which require tech companies to identify all users within a specific area at a specific time—violate the Fourth Amendment's ban on unreasonable searches by acting as a digital dragnet.

- [https://www.npr.org/2026/04/27/nx-s1-5777656/supreme-court-geofence-warrants](https://www.npr.org/2026/04/27/nx-s1-5777656/supreme-court-geofence-warrants)

## Dutch Central Bank Abandons US Cloud for Lidl's IT Arm

De Nederlandsche Bank is switching to Stackit, the cloud platform owned by Lidl's Schwarz Group, to reduce its reliance on American hyperscalers and ensure its data falls strictly under European law.

- [https://www.techzine.eu/news/infrastructure/140634/dutch-central-bank-chooses-lidl-for-european-cloud/](https://www.techzine.eu/news/infrastructure/140634/dutch-central-bank-chooses-lidl-for-european-cloud/)

## GitHub Copilot Shifts to Usage-Based Billing

Starting June 1, 2026, GitHub Copilot will replace premium request units with 'GitHub AI Credits.' While base plan prices remain the same, usage will now be calculated based on token consumption (input, output, and cached tokens).

- [https://github.blog/news-insights/company-news/github-copilot-is-moving-to-usage-based-billing/](https://github.blog/news-insights/company-news/github-copilot-is-moving-to-usage-based-billing/)

## Critical 'Pack2TheRoot' Linux Privilege Escalation

A vulnerability in the PackageKit daemon affects a wide range of Linux distributions, including Ubuntu, Debian, and Fedora. It allows any local unprivileged user to install or remove system packages without authorization to obtain full root access.

- [https://github.security.telekom.com/2026/04/pack2theroot-linux-local-privilege-escalation.html](https://github.security.telekom.com/2026/04/pack2theroot-linux-local-privilege-escalation.html)

## Original Creators Return with 'Super ZSNES'

The original developers of the legendary ZSNES emulator have returned with 'Super ZSNES,' a complete rewrite that uses GPU acceleration to add widescreen support, texture mapping, and uncompressed audio to classic SNES games.
