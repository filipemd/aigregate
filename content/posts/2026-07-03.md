## Windows 11 Now Runs Linux Containers Natively

Microsoft has released a public preview of WSL Containers, enabling Windows 11 users to build and run Linux containers without needing Docker Desktop. This removes a significant dependency for developers, offering a first-party 'front door' to containers with built-in GPU passthrough and enterprise-grade security controls via Group Policy.

## Linux 7.3 to Axe 20-Year-Old EFS File System

The EFS file-system driver, originally used for SGI IRIX systems, is slated for removal in Linux 7.3. Kernel developers decided to purge the outdated code after it had been unmaintained for 20 years, noting that the driver is virtually nonexistent in the wild since 1999.

- [https://www.phoronix.com/news/EFS-File-System-Removal-Coming](https://www.phoronix.com/news/EFS-File-System-Removal-Coming)

## Wormzy: High-Speed Secure P2P File Sharing

Wormzy is a new open-source tool designed for fast, secure peer-to-peer sharing of large files. By leveraging QUIC and Noise, it bypasses the need for NAT configuration or port forwarding, providing a modern, encrypted alternative to tools like Magic Wormhole.
